# Hex Inventions

**Daily reverse-engineered inventions from the Unseen University**

Every day at midday, the Igor Swarm wakes up and:

1. Scans what's trending in news, apps, legacy industries, and viral culture
2. Picks the one thing most likely to go viral
3. Reverse engineers it using the IGOR Protocol (Observe, Decompose, Map, Hypothesize, Verify, Transcend)
4. Designs something better - with new powers, iterations, and magicks
5. Publishes the full teardown + invention here

## Structure

Each invention lives in its own dated directory:

```
inventions/
  2026-04-03-smart-doorbell/
    README.md           # The full case study
    RE-FINDINGS.md      # What we discovered
    INVENTION.md        # What we designed
    PATENT-DRAFT.md     # UK IPO format patent application
    diagrams/           # Architecture maps, block diagrams
    src/                # Any code (if software invention)
```

## The IGOR Protocol

```
OBSERVE -> DECOMPOSE -> MAP -> HYPOTHESIZE -> VERIFY -> TRANSCEND
```

Every teardown follows six phases. We don't just understand - we **transcend**.
